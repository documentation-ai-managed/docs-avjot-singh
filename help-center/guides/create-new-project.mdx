---
title: Creating a New Project
description: Step-by-step guide for adding a new documentation project.
---

## Steps

<Steps>
  <Step title="Go to Dashboard">
    ````
    Visit https://docs.avjotsingh.com/dashboard
    ````
  </Step>
  <Step title="Click 'New Project'">
    Fill in project name and optional description.
  </Step>
  <Step title="Set Permissions">
    Invite team members if needed.
  </Step>
  <Step title="Start Documenting">
    Add pages or import content to begin.
  </Step>
</Steps>
